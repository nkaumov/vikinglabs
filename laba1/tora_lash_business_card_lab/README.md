# Лабораторная работа: «Личная визитка»

Простая консольная программа на Python в тематике студии **Tora Lash & Brow**.
Программа запрашивает имя, фамилию, год рождения и рост, вычисляет примерный возраст и выводит данные в красивой рамке.

## Как запустить

1. Установите Python 3.10 или новее.
2. Откройте терминал в папке проекта.
3. Выполните команду:

```bash
python main.py
```

Если команда `python` не работает, попробуйте:

```bash
py main.py
```

## Состав проекта

- `main.py` — исходный код программы.
- `report_tora_lash_business_card.docx` — отчет по лабораторной работе.
- `run_windows.bat` — быстрый запуск программы в Windows.
