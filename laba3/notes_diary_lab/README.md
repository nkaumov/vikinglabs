# Лабораторная работа: Дневник заметок

Простая консольная программа на Python для создания заметок с автоматической датой и временем. Данные хранятся в JSON-файле `data/notes.json`.

## Запуск в Windows PowerShell или VS Code Terminal

```bash
python main.py
```

## Запуск в Linux / GitHub Codespaces

```bash
python3 main.py
```

## Что показать на скриншотах для отчета

1. Создание новой заметки.
2. Просмотр всех заметок.
3. Поиск заметки по дате.
4. Содержимое файла `data/notes.json`.

В Linux можно дополнительно выполнить:

```bash
cat /etc/os-release
python3 --version
cat data/notes.json
```
